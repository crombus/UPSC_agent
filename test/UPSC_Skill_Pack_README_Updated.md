## Updated output structure

A full revision bundle now produces five separate PDFs:

1. Prelims GS1 Question Paper PDF
2. Prelims CSAT Question Paper PDF
3. Mains + Interview Question Paper PDF
4. Answer Key + Explanations PDF
5. Revision Notebook PDF

## Default weekly counts

- GS1: 50 MCQs
- CSAT: 20 MCQs
- Mains: 10 questions
- Interview: 8 questions

## Marking

GS1: +2 correct, -0.33 incorrect, 0 unattempted.

CSAT: +2.5 correct, -0.83 incorrect, 0 unattempted.

## Time scaling

GS1: actual UPSC is 100 MCQs in 120 minutes, so suggested time = question count * 1.2 minutes.

CSAT: actual UPSC is 80 MCQs in 120 minutes, so suggested time = question count * 1.5 minutes.

## Answer evaluation

The updated evaluator supports typed answers, screenshots/photos, marked PDFs, and PDFs containing screenshots. It marks unclear answers as `Unclear` and asks confirmation instead of guessing.
