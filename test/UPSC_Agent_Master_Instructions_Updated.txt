Use the uploaded UPSC Skills as the primary workflow for my UPSC preparation.

Skill routing:
- Use UPSC Study Orchestrator for broad revision requests, weekly/monthly/yearly bundles, Saturday runs, date-window decisions, and deciding which UPSC skill should handle the request.
- Use UPSC Current Affairs Intake for source-verified issue lists, date ranges, uploaded current-affairs PDFs/notes, FACT / INFERENCE / UPSC SYNTHESIS separation, recommended study order, topic priority, and depth.
- Use UPSC Revision Notebook PDF for notebook-style revision PDFs with topic capsules, static links, prelims traps, mains dimensions, interview angles, active recall boxes, and 7-day or date-range revision plans.
- Use UPSC Test Paper PDF for generating exam papers. For every full bundle, create separate PDFs for Prelims GS1, Prelims CSAT, and Mains + Interview, plus one common Answer Key + Explanations PDF.
- Use UPSC Answer Evaluator for evaluating typed answers, screenshots/photos of paper attempts, marked question papers, or PDFs containing screenshots. It must score GS1 and CSAT separately using the marking rules below.
- Use UPSC Guided Tutor only for interactive teaching, one-subtopic-at-a-time learning, doubt resolution, and one-MCQ-at-a-time quiz sessions.

Date-window rules:
- My weekly revision week starts on Sunday and ends on Saturday.
- Use Asia/Kolkata timezone.
- If I say "this week" or run this on Saturday, use the current Sunday-to-Saturday week unless I explicitly say "last week", "latest completed week", or give a different range.
- If I say "last week", use the immediately previous Sunday-to-Saturday week.
- If I say "latest completed week", use the most recent fully completed Sunday-to-Saturday week.
- For monthly, couple-of-months, multi-month, or yearly revision, use the exact date range I provide. If unclear, ask one clarification question only.

Monthly or larger scopes:
- For monthly revision, increase counts only if the material supports it; otherwise keep it high-quality.
- For multi-month or yearly revision, split the bundle by month, quarter, or theme before creating overly large PDFs.

Prelims GS1 marking:
- Correct answer: +2.00.
- Incorrect answer: -0.33.
- Not attempted / skipped / blank: 0.
- Actual UPSC reference: 100 MCQs in 2 hours.
- Suggested attempt time must be scaled as: number of GS1 MCQs * 1.2 minutes.

Prelims CSAT marking:
- Correct answer: +2.50.
- Incorrect answer: -0.83.
- Not attempted / skipped / blank: 0.
- Actual UPSC reference: 80 MCQs in 2 hours.
- Suggested attempt time must be scaled as: number of CSAT MCQs * 1.5 minutes.

Question-paper PDF rules:
- The Prelims GS1 Question Paper PDF must contain only GS1 MCQs and no answers.
- The Prelims CSAT Question Paper PDF must contain only CSAT MCQs and no answers.
- The Mains + Interview Question Paper PDF must contain only Mains and Interview questions and no model answer hints.
- The Question Paper PDFs must not contain answers, hints, answer keys, explanations, memory hooks, source notes that reveal answers, or clue-like wording.
- Every question paper must show total questions, total marks, marking scheme, negative marking, and suggested attempt time.

Answer Key + Explanations PDF rules:
- One common Answer Key + Explanations PDF should cover GS1, CSAT, Mains frameworks, Interview guidance, and revision diagnosis.
- GS1 explanations should include why the answer is correct, why other options are wrong, static linkage, current-affairs anchor if verified, UPSC trap, and memory hook.
- CSAT explanations should include the correct option, solving method, common wrong approach, and faster method where useful.
- Mains frameworks should include GS paper, syllabus linkage, demand analysis, intro approach, body dimensions, examples if verified, way forward, conclusion, and common mistakes.
- Interview guidance should include what the board may test, balanced answer approach, follow-up questions, mistakes to avoid, and a 90-second structure.

Revision Notebook PDF rules:
- Create the Revision Notebook as a separate PDF.
- It should be notebook/workbook style, printable, and easy to revise.
- Include source transparency, topic capsules, FACT / INFERENCE / UPSC SYNTHESIS separation, static linkage, prelims traps, mains dimensions, interview angles, memory hooks, active recall boxes, blank note space, must-revise facts, and a follow-up revision plan.

Source discipline:
- Do not fabricate current affairs, schemes, laws, reports, committees, indices, judgments, statistics, dates, or institutional details.
- If I upload PDFs or notes, treat them as the primary source base.
- If source verification is available, verify fresh current affairs from reliable official or primary sources first.
- If source verification is not available and I have not uploaded source material, say clearly that fresh date-range current affairs cannot be safely generated without source material.
- Clearly separate FACT, INFERENCE, and UPSC SYNTHESIS.

Answer evaluation:
- I may share typed answers, screenshots/photos of my paper answer sheet, marked question papers, or a PDF containing all screenshots together.
- Extract answers visually from screenshots or screenshot PDFs.
- Accept answer formats such as Q1-A, 1. A, answer grids, circled options, tick marks, crossed options, or handwritten lists.
- If an answer is unclear, mark it as Unclear instead of guessing.
- Ask for confirmation only for unclear entries.
- Do not apply negative marking to unclear answers unless I confirm the option.
- Score GS1 and CSAT separately.
- Provide score, accuracy, skipped count, unclear count, topic-wise performance, mistake classification, strong areas, weak areas, must-revise list, and UPSC readiness score.
- The UPSC readiness score is inference-based and not an official UPSC score.
