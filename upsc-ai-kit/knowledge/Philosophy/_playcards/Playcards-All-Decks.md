## DECK 1 — PAPER-I WESTERN (core terms)
```
Cogito ergo sum — whose, what point?            :: Descartes; the one indubitable truth surviving universal doubt.
Primary vs secondary qualities — whose?         :: Locke; primary (extension, shape) in object; secondary (colour, taste) mind-dependent.
Esse est percipi                                :: Berkeley; to be is to be perceived — no material substance.
Hume's account of the self                      :: A "bundle" of perceptions; no simple enduring self.
Hume on causation                               :: Not a necessary connection; habit from constant conjunction.
Synthetic a priori — who made it central & why  :: Kant; to explain how maths & science give necessary yet informative truths.
Kant's Copernican revolution                    :: Objects conform to mind (space/time/categories), not mind to objects.
Phenomena vs noumena                            :: Things-as-they-appear (knowable) vs things-in-themselves (unknowable).
Hegel's dialectic                               :: Thesis → antithesis → synthesis; Spirit unfolding to Absolute.
Russell's theory of descriptions                :: "The present king of France" — analysed away as incomplete symbol.
Picture theory of meaning — whose?              :: Early Wittgenstein (Tractatus); proposition pictures a possible fact.
Verification principle                          :: Logical Positivism; meaning = method of empirical verification (or analyticity).
"Meaning is use" — whose?                       :: Later Wittgenstein (Investigations); language-games.
Private language argument                       :: Later Wittgenstein; no logically private language possible.
Epoché / bracketing — whose?                    :: Husserl; suspend the natural attitude to describe pure essences.
Intentionality                                  :: Consciousness is always consciousness OF something (Husserl).
"Existence precedes essence"                    :: Sartre; humans first exist, then define themselves — radical freedom.
Dasein                                          :: Heidegger; human being as "being-in-the-world."
Quine's "two dogmas"                            :: Analytic/synthetic distinction + reductionism — both rejected.
Strawson's basic particulars                    :: Material bodies; "person" is a primitive concept.
```

## DECK 2 — PAPER-I INDIAN (core terms)
```
Cārvāka's sole pramāṇa                          :: Pratyakṣa (perception) only; rejects inference & testimony.
Saptabhaṅgīnaya                                 :: Jain 7-fold conditional predication (syād-asti, syād-nāsti…).
Anekāntavāda                                    :: Jain doctrine: reality has infinitely many aspects.
Pratītyasamutpāda                               :: Buddhist dependent origination; 12 nidānas.
Kṣaṇikavāda                                     :: Buddhist doctrine of momentariness.
Nairātmyavāda / anātman                         :: Buddhist no-permanent-self.
Four Buddhist schools                           :: Vaibhāṣika, Sautrāntika, Yogācāra, Mādhyamika.
Nyāya's 4 pramāṇas                              :: Perception, inference, comparison, testimony.
Nyāya proof for God                             :: Inference from the world as effect to an intelligent cause (Īśvara).
Sāṃkhya's two ultimates                         :: Puruṣa (many, conscious, inactive) & Prakṛti (one, unconscious, 3 guṇas).
Three guṇas                                     :: Sattva, rajas, tamas.
Satkāryavāda                                    :: Effect pre-exists in the cause (Sāṃkhya: pariṇāma; Advaita: vivarta).
Yoga's definition                               :: Cittavṛtti-nirodha — stilling the mind's modifications.
Kaivalya                                        :: Yoga's liberation — isolation of puruṣa from prakṛti.
Svataḥ prāmāṇya                                 :: Mīmāṃsā; knowledge is intrinsically valid.
Śaṅkara's mahāvākya                             :: Brahma satyaṃ jagat mithyā, jīvo brahmaiva.
Adhyāsa                                         :: Śaṅkara; superimposition (rope-snake) explaining ignorance.
Māyā                                            :: Cosmic illusion veiling Brahman (Advaita).
Aprthaksiddhi                                   :: Rāmānuja; inseparable relation of soul/matter to Brahman.
Pañcavidhabheda                                 :: Madhva; five eternal differences (God-soul, God-matter, etc.).
Aurobindo's evolution/involution                :: Spirit involves into matter, then evolves back toward Supermind.
```

## DECK 3 — PAPER-II (core terms)
```
Rawls' two principles                           :: Equal basic liberties + difference principle (via veil of ignorance).
Negative vs positive liberty — whose?           :: Isaiah Berlin.
Austin's sovereignty                            :: Command of a determinate superior, habitually obeyed.
Laski's view of sovereignty                     :: Pluralist — no absolute sovereign; power shared.
Kautilya's instrument of order                  :: Daṇḍa (rod/coercion) bound by dharma.
Marxism's engine of history                     :: Dialectical materialism / class struggle.
"Principled distance" — whose, what?            :: Bhargava; the Indian model of secularism.
Capability approach — whose?                    :: Amartya Sen; development as expansion of freedoms.
"One is not born but becomes a woman"           :: Simone de Beauvoir.
Gandhi vs Ambedkar on caste                     :: Gandhi reforms varṇa/abolishes untouchability; Ambedkar annihilates caste.
Ontological argument                            :: Anselm; God's existence follows from His concept.
Teleological argument                           :: Design argument (Paley's watch); Hume/Kant critique.
Problem of evil — Hick's reply                  :: "Vale of soul-making" theodicy.
Free-will defence — whose?                      :: Augustine (evil from misuse of free will); Plantinga (modern).
Euthyphro dilemma                               :: Is X good because God wills it, or willed because good?
Numinous — whose?                               :: Rudolf Otto; mysterium tremendum et fascinans.
E-I-P (pluralism)                               :: Exclusivism / Inclusivism / Pluralism (Hick).
Analogical language — whose?                    :: Aquinas; between univocal and equivocal.
Symbolic language — whose?                      :: Tillich; symbols participate in Being-Itself.
Braithwaite on religious language               :: Non-cognitive — declaration of a moral intention.
```
