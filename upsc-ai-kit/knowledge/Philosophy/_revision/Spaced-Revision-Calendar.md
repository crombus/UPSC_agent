# SPACED-REVISION CALENDAR — D1 / D7 / D30 + pre-exam sprint

> **Use:** every topic file, once studied, enters this cycle. Retention beats re-reading. ✅ method.

---

## THE CYCLE (per topic)
```
STUDY (Day 0) → D1 (playcards, cover-recall) → D7 (master-sheet line + 1 PYQ skeleton)
             → D30 (full active recall + write one 10-marker) → then monthly touch
```

| Interval | What to do | Time | Material |
|---|---|---|---|
| **D0 study** | read topic file, make the visual/mnemonic your own | 45–60 min | topic `.md` |
| **D1** | playcards for that topic; recall aloud | 10 min | `_playcards/` |
| **D7** | recite master-sheet line; write ONE model-answer skeleton from memory | 20 min | Master-Sheet + PYQ bank |
| **D30** | full recall; write one timed 10-marker; self-mark vs skeleton | 30 min | topic file + `_application/` |
| **Monthly** | scan master sheet + comparison matrix only | 15 min | `_revision/` |
