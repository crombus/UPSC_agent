## 1. Architecture

```text
NOLAN PRINCIPLES (aspirational floor)         DILEMMA TYPES (where principles collide)
Selflessness, Integrity, Objectivity,          Type 1: Duty vs Duty
Accountability, Openness, Honesty,              (loyalty to superior vs public interest)
Leadership                                     Type 2: Value vs Value
        |                                       (compassion vs impartiality)
        v                                       Type 3: Individual vs Institutional
   CODE OF ETHICS                                (personal conscience vs organisational decision)
   (broad, aspirational)                        Type 4: Short-term vs Long-term
        |                                       (immediate relief vs sustainable/systemic solution)
        v
   CODE OF CONDUCT
   (precise, sanctionable)
```

**Analytical claim:** The Nolan principles function as an aspirational *ceiling* while the Code of
Conduct functions as an enforceable *floor* — genuine ethical dilemmas live in the gap between
them, where conduct is technically lawful but the Nolan-standard resolution is contested.

## 2. Concepts and distinctions

| Concept | Precise meaning |
|---|---|
| ✅ **Selflessness vs self-interest bias** (Nolan Principle 1) | Requires decisions "solely in terms of public interest" — the most demanding principle because self-interest can be subtle (reputational, career, ideological) not just financial. |
| ✅ **Objectivity in Nolan's specific sense** | Confined to public appointments, contracts, and awarding of benefits — narrower than a general "be unbiased" injunction; it is procedural (merit-based selection), not a claim about value-neutrality in policy itself. |
| ⚠️ **Nested dilemmas** | Complex administrative situations often contain more than one dilemma simultaneously (e.g., a disaster-relief case combining resource-allocation fairness, personal-versus-duty conflict, and transparency-versus-order-maintenance) — advanced answers should decompose and address each nested dilemma distinctly. |
| ⚠️ **Genuine vs manufactured dilemma** | A "manufactured" dilemma disguises a simple rule violation as a values conflict (e.g., presenting bribery as "compassionate flexibility") — an advanced skill is distinguishing real value conflicts from rule violations dressed up as dilemmas. |

## 3. Detailed treatment

### The seven Nolan principles — precise, non-overlapping readings

- ✅ **Selflessness:** decisions "solely" in public interest, not "primarily" — the ARC's citation
  (2.2.5) uses the strong term, useful for identifying even partial self-interest as a violation.
- ✅ **Integrity:** freedom from financial/other obligation that "might influence" official duty —
  note this is a *risk-based* standard (potential influence), not requiring proof of actual bias,
  which is why disclosure/recusal regimes exist even absent demonstrated wrongdoing.
- ✅ **Accountability:** "must submit... to whatever scrutiny is appropriate to their office" — an
  explicitly submission-based, externally-facing standard, distinguishing it from mere internal
  self-assessment (see `11` for accountability vs responsibility).
- ✅ **Openness:** "as open as possible," restricting information "only when the wider public
  interest clearly demands" — this reverses the default; secrecy requires justification, not
  disclosure (directly underpinning RTI's presumption of disclosure, see `15`).
- ✅ **Honesty:** specifically about *declaring private interests* and resolving resulting conflicts
  to protect public interest — narrower than a general truthfulness norm; it targets conflict-of-
  interest management specifically.
- ✅ **Leadership:** promoting the principles "by leadership and example" — makes senior officials
  responsible not just for their own conduct but for modelling standards that shape subordinates'
  and institutional culture.

### Distinguishing genuine dilemmas from manufactured ones

- ⚠️ A genuine dilemma survives even after removing all self-interest and rule-violation from the
  scenario — e.g., choosing between two lawful, legitimate but competing policy priorities
  (economic development vs environmental protection in a specific project) is a genuine dilemma.
- ⚠️ A manufactured dilemma dissolves once self-interest or rule-violation is named plainly — e.g.,
  "should I accept this gift from a contractor to maintain a good working relationship" is not a
  genuine dilemma; it is a conflict-of-interest violation framed euphemistically as one.
- ⚠️ GS-IV case studies (see `22`) deliberately blend both types; a strong answer explicitly tests
  and states which type it is facing before proceeding to resolution.

### Nested dilemmas in complex cases

- ⚠️ The 2025 GS-IV DC-Vijay case study nests at least three distinct dilemmas: (1) personal grief
  vs public duty (an emotional-intelligence dilemma, see `05`), (2) resource-allocation fairness in
  disaster relief (a distributive-justice dilemma, see `08`'s Rawls/utilitarian tension), and
  (3) transparency vs maintaining public calm during the crisis (an openness-Nolan-principle
  dilemma) — advanced answers decompose and resolve each nested layer rather than treating the case
  as a single monolithic problem.

## 4. Institutional and reform architecture

- ✅ ARC 2.2.6's Code of Ethics/Code of Conduct distinction operationalises the Nolan principles:
  the Code of Ethics states the aspirational Nolan-type standard; the Code of Conduct converts the
  narrower, more objectively verifiable elements (gift limits, disclosure timelines, post-retirement
  cooling-off) into sanctionable rules (see `16` for the full ARC recommendations on Ministers' and
  legislators' codes).
- ⚠️ India's continuing gap (Code of Conduct largely in place; comprehensive Code of Ethics still
  incomplete for many public-office categories) is directly tested — 2025 GS-IV Q5(a) asks
  candidates to "suggest a suitable model for code of ethics" precisely because the ARC's 2007
  recommendation for a Ministers' Code of Ethics has not been comprehensively implemented.

## 5. Indian applications and boundary cases

- ⚠️ A civil servant recusing themselves from a tender evaluation involving a relative's firm
  operationalises Nolan's integrity principle even absent any proven wrongdoing — the risk-based
  standard justifies precautionary recusal.
- ⚠️ Boundary case: an official disclosing sensitive but non-security-critical information under RTI
  despite departmental reluctance operationalises Nolan's openness principle (secrecy requires
  justification) — but must still respect legitimate exemptions (security, ongoing investigation)
  under law (see `15`).

## 6. Limitations and trade-offs

- ⚠️ The Nolan principles, being aspirational, carry no direct sanction; without a linked Code of
  Conduct they risk becoming what the ARC (2.4.3) itself calls "not comprehensive... a list of
  prohibitions" if left unaccompanied by ethics guidance, or purely rhetorical if left unaccompanied
  by enforceable rules.
- ⚠️ Distinguishing genuine from manufactured dilemmas requires judgment that can itself be biased —
  officials under real pressure may sincerely (but wrongly) perceive a rule violation as a
  legitimate values conflict, a risk that institutional checks (peer review, vigilance oversight)
  must guard against.

## 7. Must-Know Facts for Advanced Prelims

- ✅ The Nolan Committee's formal name is the UK Committee on Standards in Public Life (established
  1994, seven principles issued 1995).
- ✅ ARC's own recommendation (2.4.5) was that a Code of Ethics be created *in addition to* the
  existing Code of Conduct for Ministers, monitored by dedicated units in the PM's/CM's offices with
  an annual report to the legislature.
- ✅ 2025 GS-IV Q5(a) explicitly acknowledges that India's Code of Conduct is "already in operation"
  while a Code of Ethics remains "yet to be put in place" — matching the ARC's own 2007 diagnosis,
  still substantially unresolved as of the current PYQ.

## 8. Advanced Prelims traps

- ❌ Nolan's "objectivity" principle requires value-neutral policymaking generally. -> It is
  specifically about merit-based decisions in appointments, contracts and benefits — a narrower,
  procedural standard.
- ❌ A genuine ethical dilemma always involves two "equally good" options. -> Options can be
  legitimately unequal in ethical weight; the dilemma is that both options honour some legitimate,
  competing duty, not that they are equivalent in merit.
- ❌ The ARC's Code of Ethics recommendation for Ministers has been fully implemented. -> The 2025
  GS-IV paper itself confirms the Code of Ethics remains "yet to be put in place," matching the
  ARC's 2007, still-largely-unaddressed recommendation.

## 11. Mains-ready framework

**Central thesis:** Ethical dilemmas in Indian public service occupy the gap between an aspirational
Nolan-type Code of Ethics and an enforceable Code of Conduct; resolving them requires first testing
whether the dilemma is genuine or manufactured, then decomposing any nested sub-dilemmas, and
finally checking the proposed resolution against all seven Nolan principles for public
defensibility.

1. State the dilemma(s) precisely; test genuine vs manufactured.
2. Decompose nested dilemmas if more than one competing-duty pair exists.
3. Apply the relevant Nolan principle(s) as an aspirational checklist.
4. Apply a moral-theory triangulation (`08`) to each sub-dilemma.
5. Propose a resolution that is defensible under external scrutiny (Nolan's accountability test).
6. Note the current Code of Ethics/Code of Conduct implementation gap where relevant.
